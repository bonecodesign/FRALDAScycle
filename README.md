# FRALDAScycle

FraldaCycle é uma plataforma para compra, venda e doação de pacotes fechados de fraldas.

## Status

🚧 Em desenvolvimento.

## Visão

Construir um ecossistema seguro para famílias, facilitando a circulação de pacotes fechados de fraldas.

## Escopo

- Compra de pacotes fechados
- Venda de pacotes fechados
- Doação de pacotes fechados
- Busca por ofertas próximas

> O projeto não realiza lavagem, higienização, reciclagem ou recondicionamento de fraldas.
